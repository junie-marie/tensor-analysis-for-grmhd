# Tensor Analysis for GRMHD Simulation
This repository focuses on building a custom tensor-analysis engine (using C + MPI) to assist in general-relativistic magnetohydrodynamics (GRMHD) simulation of jets in active galactic nuclei (AGNs).

This project is part of the High Performance Computing (CS-4/6373) course at the University of Tulsa during the Spring 2026 semester.
